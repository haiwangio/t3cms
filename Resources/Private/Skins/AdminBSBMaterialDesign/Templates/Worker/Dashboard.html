You can have this look in your BE-Modules!

Just add these layout root path for t3cms be module layout:

# PageTS
module.tx_yourcoolextension {
    view {
        layoutRootPaths.0 = EXT:your_cool_extension/Resources/Private/Layouts/
        layoutRootPaths.10 = EXT:t3cms/Resources/Private/Skins/AdminBSBMaterialDesign/Layouts/
    }
}

Check out the static demo to explore, what elements are available with this be module layout. Open static demo. Note: The t3cms backend module layout is a very slim fork of the original template.
